# dkarpov.is-a.dev

Personal portfolio of Daniil Karpov — Python developer & MGIMO international economics student (risk & insurance).

Built with React + Vite + Framer Motion. Bilingual (RU/EN), dark theme, custom animations.

## Development

```bash
npm install
npm run dev
```

## Deployment

Pushes to `main` are built and deployed to GitHub Pages automatically via GitHub Actions.
Custom domain: [dkarpov.is-a.dev](https://dkarpov.is-a.dev)
